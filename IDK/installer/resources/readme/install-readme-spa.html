Resumen del IDK

Breve Descripción

El INGENIAS Development Kit (IDK) permite a los diseñadores crear Sistemas Multi-Agentes (SMA) con un aproximación basada en el Desarrollo del Software Dirigido por Modelos (DSDM). Es decir, los diseñadores crean una especificiación del SMA con el editor del IDK, y los generadores de código (por el momento principalmente el generador INGENIAS Agent Framework para la plataforma JADE) crean un esqueleto del código de programación del SMA.

La Versión que estás instalando.

Cada año se suele hacer una versión estable del IDK. Sin embargo, el IDK está en continuo desarrollo. Este instalador contiene las últimas fuentes del IDK actualizadas el 8 de octubre del 2009.
