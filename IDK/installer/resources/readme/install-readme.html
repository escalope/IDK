IDK Overview

What it can do for you

The INGENIAS Development Kit (IDK) allows designers to create Multi-Agent Systems (MASs) with a Model-Driven Development (MDD) approach. In other words, designers create an specification of the MAS with the IDK editor tool, and the code generators (for now mainly the INGENIAS Agent Framework generator for JADE platform) create an skeleton of the programming code of the MAS.

The version you are installing

Each year a stable version is usually released for the IDK. However, the IDK is continuoslly evolving. This installer contains the latest sources of the IDK up to October 8th, 2009.
